# belleangelina.github.io

Astro 静态文章展示站，发布到 GitHub Pages：`https://belleangelina.github.io/`。

当前站点名称：`天然未来派的摸鱼小屋`。

## 常用命令

```bash
npm run dev
npm run check
npm run build
npm run preview
npm run verify
```

`npm run verify` 会依次执行 Astro 类型检查、静态构建和产物审计，是提交前的完整验证入口。
